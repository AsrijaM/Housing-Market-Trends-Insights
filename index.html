1. Data Collection & Extraction from Database (SQL)

To feed our Tableau workbook extracts efficiently, we write an optimized SQL script that computes complex metrics directly on the server level, minimizing dashboard calculations. 

-- Precompute Age and Renovation Categories at DB layer
SELECT 
    id, sale_date, price, bedrooms, bathrooms, floors,
    (EXTRACT(YEAR FROM sale_date) - yr_built) AS house_age,
    CASE WHEN yr_renovated > 0 THEN 1 ELSE 0 END AS is_renovated,
    CASE 
        WHEN yr_renovated > 0 THEN (EXTRACT(YEAR FROM sale_date) - yr_renovated)
        ELSE (EXTRACT(YEAR FROM sale_date) - yr_built)
    END AS years_since_renovation,
    CASE 
        WHEN yr_renovated = 0 THEN 'Never Renovated'
        WHEN (EXTRACT(YEAR FROM sale_date) - yr_renovated) <= 5 THEN 'Recently Renovated (<5 yrs)'
        ELSE 'Long-term Renovated (>15 yrs)'
    END AS renovation_category
FROM staging_house_sales
WHERE price > 0 AND bedrooms IS NOT NULL;
2. Data Preparation & Outlier Cleaning (Python)

We run a secondary data profiling phase in Python to remove architectural structural outliers (e.g. houses with 33 bedrooms but 1 bathroom) and fill in missing zipcode records before staging to CSV. 

import pandas as pd
import numpy as np

# Load & Clean Outliers
df = pd.read_csv('raw_house_sales.csv')

# Handle missing bedroom and bathroom data
df['bedrooms'] = df['bedrooms'].fillna(df['bedrooms'].median()).astype(int)
df['bathrooms'] = df['bathrooms'].fillna(df['bathrooms'].median())

# Calculate house age and years since renovation
df['sale_year'] = pd.to_datetime(df['sale_date']).dt.year
df['house_age'] = (df['sale_year'] - df['yr_built']).clip(lower=0)
df['is_renovated'] = (df['yr_renovated'] > 0).astype(int)
df['years_since_renovation'] = np.where(df['yr_renovated'] > 0, df['sale_year'] - df['yr_renovated'], df['house_year'] - df['yr_built'])

# Export clean extract for Tableau
df.to_csv('clean_house_sales.csv', index=False)
